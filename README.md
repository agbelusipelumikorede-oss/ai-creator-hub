# AI Creator Hub

AI-powered platform to help creators generate content ideas, captions, scripts, and schedule posts. Subscription-based.

## Features
- AI-generated content ideas
- AI-generated captions/scripts
- Post scheduling
- Free & Pro subscription tiers

## Tech Stack
- Next.js + TypeScript
- Tailwind CSS
- NextAuth.js (authentication)
- Prisma + PostgreSQL (database)
- OpenAI API (AI content generation)
- Stripe (subscription payments
